Syn攻擊方法：
丟Syn封包(亂數來源IP)，不回傳ACK，讓對方一直丟 SYN-ACK ，耗盡其資源

1.掃描目標主機開啟的 port
2.針對每個掃到的 port 開多個 thread
3.在 thread 中針對其port狂丟Syn封包
